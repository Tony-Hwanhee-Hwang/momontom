# Momontom
Clone coding chrome extension "Momentom" with vanilla JS

# Features
- [x] Clock
- [x] To do List
- [x] Weather with Geolocation
- [x] Random background
- [x] Username Persistance
- [x] Offline Todo list and username support using local storage
